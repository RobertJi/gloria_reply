Facebook Webhook Service

About This Service

This is a Node.js service designed to handle Facebook webhook events. It provides endpoints for webhook verification and event processing, making it easy to integrate with Facebook's API.

Available Endpoints

GET /webhook

Handles Facebook webhook verification

POST /webhook

Processes incoming Facebook webhook events

GET /health

Health check endpoint

Quick Start

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env

# Start the service
npm run dev
